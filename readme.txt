Запуск программы:
go run main.go

Файл с API-ключами от Ankr:
./config/api_keys.txt
Добавлять ключи БЕЗ запятых, каждый ключ с новой строки

Файл основных настроек (конфиг):
./config/setup.yaml

Результаты работы записываются в логи:
./logs/empty.log - для пустых кошельков (можно отключить в конфиге - save_empty: false)
